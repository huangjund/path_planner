Generate a path from an initial configuration to a target configuration, with a specified maximum turning radii

A configuration is (x, y, theta), where theta is in radians, with zero along the line x = 0, and counter-clockwise is positive

Parameters
q0- a configuration specified as an array of x, y, theta
q1- a configuration specified as an array of x, y, theta
rho- turning radius of the vehicle (forward velocity divided by maximum angular velocity)
path- the resultant path
Returns
- non-zero on error
